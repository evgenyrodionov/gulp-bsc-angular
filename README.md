## Clone
  
    git clone https://github.com/theaqua/gulp-bsc-angular.git myNewProject
    cd myNewProject
    rm -rf .git

## Install

    npm install -g gulp bower
    npm install && bower install

## Run
    
    gulp watch

## Build
    
    gulp build --env production
